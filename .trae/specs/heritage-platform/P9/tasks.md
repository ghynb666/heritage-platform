# P9 - 前台-非遗详情与传承人主页

本阶段包含非遗项目详情展示、搜索筛选和传承人主页功能。

---

## Task 9.1: 非遗项目详情后端

- [ ] 实现项目详情查询接口（含历史渊源、技艺特点、图片、视频）
- [ ] 实现项目搜索接口（关键词模糊搜索）
- [ ] 实现项目分类筛选功能
- [ ] 实现项目排序接口（点击量、收藏数、点赞数，升序/降序）
- [ ] 实现项目审核接口（管理员）

---

## Task 9.2: 前端非遗详情页面

- [ ] 实现非遗详情页面布局
- [ ] 实现历史渊源展示区
- [ ] 实现技艺特点展示区
- [ ] 实现高清大图展示区（支持图片预览）
- [ ] 实现制作视频播放区（在线播放）
- [ ] 实现发布者信息展示（管理员发布不显示传承人信息，传承人发布显示头像用户名）

---

## Task 9.3: 传承人主页

- [ ] 实现传承人主页布局
- [ ] 展示传承人基本信息
- [ ] 展示该传承人发布的所有非遗项目
- [ ] 实现点击传承人头像/用户名跳转传承人主页

---

## Task 9.4: 搜索排序功能

- [ ] 实现排序功能（点击量、收藏数、点赞数，带升序/降序箭头）
- [ ] 实现顶部搜索框（分类筛选+关键词搜索）

---

## Task Dependencies

- Task 9.1 可独立执行
- Task 9.2 依赖于 Task 9.1
- Task 9.3 依赖于 Task 9.1
- Task 9.4 依赖于 Task 9.1
