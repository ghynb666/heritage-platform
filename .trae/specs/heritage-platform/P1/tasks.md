# P1 - 非遗展示模块

本阶段包含非遗分类管理、非遗项目展示、详情页、互动功能模块。

---

## Task 1.1: 非遗分类管理后端

- [ ] 创建非遗分类实体类和Mapper（剪纸、刺绣、陶艺、皮影、木雕等具体分类）
- [ ] 实现分类增删改查接口
- [ ] 实现分类树形结构查询
- [ ] 实现分类排序接口

---

## Task 1.2: 非遗项目管理后端

- [ ] 创建非遗项目实体类和Mapper（包含历史渊源、技艺特点字段）
- [ ] 创建项目图片实体类和Mapper（支持多张高清大图）
- [ ] 创建项目视频实体类和Mapper（支持在线播放）
- [ ] 实现项目发布接口（传承人，限制只能发布自己所属类型）
- [ ] 实现项目发布接口（管理员，不限类型）
- [ ] 实现项目列表分页查询接口
- [ ] 实现项目详情查询接口（含历史渊源、技艺特点、图片、视频）
- [ ] 实现项目搜索接口（关键词模糊搜索）
- [ ] 实现项目审核接口（管理员）
- [ ] 实现项目排序接口（点击量、收藏数、点赞数，升序/降序）

---

## Task 1.3: 文件上传功能

- [ ] 实现图片上传接口（支持多图上传）
- [ ] 实现视频上传接口
- [ ] 配置文件存储路径
- [ ] 实现图片/视频访问静态资源映射
- [ ] 实现视频在线播放支持

---

## Task 1.4: 前端非遗展示页面

- [ ] 实现非遗展示页面布局（左侧分类栏、右侧项目列表）
- [ ] 实现分类导航组件（使用具体非遗类型名称）
- [ ] 实现项目列表展示（缩略图、名称、发布者信息）
- [ ] 实现排序功能（点击量、收藏数、点赞数，带升序/降序箭头）
- [ ] 实现顶部搜索框（分类筛选+关键词搜索）
- [ ] 实现"详情"按钮（不使用"阅读更多"）

---

## Task 1.5: 前端非遗详情页面

- [ ] 实现历史渊源展示区
- [ ] 实现技艺特点展示区
- [ ] 实现高清大图展示区（支持图片预览）
- [ ] 实现制作视频播放区（在线播放）
- [ ] 实现评论互动区
- [ ] 实现发布者信息展示（管理员发布不显示传承人信息，传承人发布显示头像用户名）
