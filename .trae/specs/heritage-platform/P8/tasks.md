# P8 - 管理员后台-数据统计

本阶段包含数据统计仪表盘和敏感词管理功能。

---

## Task 8.1: 数据统计后端

- [ ] 实现用户统计接口（用户总数、新增用户、传承人数量）
- [ ] 实现非遗项目统计接口（项目总数、各分类数量）
- [ ] 实现商品统计接口（商品总数、销量统计）
- [ ] 实现交易统计接口（订单总数、交易金额）
- [ ] 实现访问统计接口（访问量、热门内容）
- [ ] 实现趋势数据接口（按日/周/月统计）

---

## Task 8.2: 敏感词管理后端

- [ ] 创建敏感词实体类和Mapper
- [ ] 实现敏感词列表查询接口
- [ ] 实现敏感词增删改查接口
- [ ] 实现敏感词批量导入接口

---

## Task 8.3: 前端数据统计页面

- [ ] 实现数据统计仪表盘布局
- [ ] 实现统计卡片组件（用户、项目、商品、交易）
- [ ] 实现图表组件（趋势图、分类占比图）
- [ ] 实现热门内容排行展示

---

## Task 8.4: 前端敏感词管理页面

- [ ] 实现敏感词列表页面
- [ ] 实现敏感词增删改功能
- [ ] 实现敏感词批量导入功能

---

## Task Dependencies

- Task 8.1 和 Task 8.2 可并行执行
- Task 8.3 依赖于 Task 8.1
- Task 8.4 依赖于 Task 8.2
