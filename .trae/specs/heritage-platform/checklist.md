## 核心验收要点

### 角色与权限
- [x] 三种角色（普通用户、传承人、管理员）权限分离正确
- [x] 传承人必须经过管理员审核才能发布内容
- [x] 传承人只能发布自己所属类型的内容

### 管理员后台
- [x] 一级菜单精简（用户管理、非遗项目管理、文创管理、资讯管理）
- [x] 传承人审核功能完整（查看资质证明、证书）
- [x] 用户管理功能完整（列表、禁用、删除、重置密码）
- [x] 传承人管理功能完整（列表、修改类型、取消资格）
- [x] 地区管理功能完整（树形展示、增删改）
- [ ] 内容审核功能完整（非遗项目、商品、资讯）
- [ ] 数据统计仪表盘展示正确

### 非遗展示
- [x] 分类栏在页面左侧，使用具体非遗类型名称
- [x] 排序功能显示升序/降序箭头指示

- [ ] 视频可在线播放
- [ ] 管理员发布的内容不显示传承人信息
- [ ] 传承人发布的内容显示头像用户名，可跳转主页

### 文创商品
