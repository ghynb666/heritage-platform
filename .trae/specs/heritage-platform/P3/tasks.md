# P3 - 非遗分类与项目展示

本阶段包含非遗分类管理、非遗项目基础展示和文件上传功能。

---

## Task 3.1: 非遗分类管理后端

- [x] 创建非遗分类实体类和Mapper（剪纸、刺绣、陶艺、皮影、木雕等具体分类）
- [x] 实现分类增删改查接口
- [x] 实现分类排序接口

---

## Task 3.2: 非遗项目管理后端

- [x] 创建非遗项目实体类和Mapper（包含历史渊源、技艺特点字段）
- [x] 创建项目图片实体类和Mapper（支持多张高清大图）
- [x] 创建项目视频实体类和Mapper（支持在线播放）
- [x] 实现项目发布接口（传承人，限制只能发布自己所属类型）
- [x] 实现项目发布接口（管理员，不限类型）
- [x] 实现项目列表分页查询接口

---

## Task 3.3: 文件上传功能

- [x] 实现图片上传接口（支持多图上传）
- [x] 实现视频上传接口
- [x] 配置文件存储路径
- [x] 实现图片/视频访问静态资源映射
- [x] 实现视频在线播放支持

---

## Task 3.4: 前端非遗展示页面

- [x] 实现非遗展示页面布局（左侧分类栏、右侧项目列表）
- [x] 实现分类导航组件（使用具体非遗类型名称）
- [x] 实现项目列表展示（缩略图、名称、发布者信息）
- [x] 实现"详情"按钮（不使用"阅读更多"）

## Task Dependencies

- Task 3.1 和 Task 3.2 可并行执行
