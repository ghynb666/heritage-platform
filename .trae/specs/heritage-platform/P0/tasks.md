# P0 - 核心基础模块

本阶段包含项目初始化、用户认证授权和传承人审核模块，是整个系统的基础。

---

## Task 0.1: 后端项目初始化

- [ ] 创建SpringBoot项目，配置Maven依赖
- [ ] 配置application.yml（数据库连接、文件上传等）
- [ ] 集成MyBatis-Plus、Swagger/Knife4j
- [ ] 配置统一返回结果封装类
- [ ] 配置全局异常处理器

---

## Task 0.2: 前端项目初始化

- [ ] 使用Vue CLI/Vite创建Vue3项目
- [ ] 安装Element Plus组件库
- [ ] 配置路由vue-router
- [ ] 配置状态管理Pinia
- [ ] 配置axios请求封装

---

## Task 0.3: 数据库设计与初始化

- [ ] 创建数据库heritage_platform
- [ ] 编写并执行数据库初始化脚本（用户表、角色表、传承人申请表等）
- [ ] 创建测试数据（管理员账号、测试用户、非遗分类等）

---

## Task 0.4: 用户实体与数据层

- [ ] 创建用户相关实体类（User、Role、UserRole）

- [ ] 创建用户相关Mapper接口和XML映射文件
- [ ] 编写用户基础CRUD方法

## Task 0.6: 角色权限管理
